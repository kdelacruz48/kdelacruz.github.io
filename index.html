My Work

Term Scheduler - Xamarin Forms

Created with college students in mind, this program utilizes xamarin Forms to endable the user to install on either android or ios. It allows students to input all information for their current term and set reminders for assignments and tests. programmed using C# and Visual studio, using a SQLite database for storage.

Appointment Manager - Windows Forms

Created for medical proffesionals, made using Windows Forms. The appointmnet manager allows the user to create, update, and delete customer and appointment data, supports multiple users, and demonstrates the ability to changed languages based on user region. It also allows the user search the databsase for reports and customer appointments. Programmed using C# and uses a MySQL database to store customer and appointment data.

Inventory Tracker- Windows Forms

Created to help track inventory, made using Windows Forms. The Inventory trackers allows the user to input various parts, and products, as well as search throught them. The parts can be labeled as either in-house or out-sourced and all parts can be added to a specific product so that parts can be tracked.
